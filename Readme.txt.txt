To run the VNTS algorithm follow the steps:
	1. Install 'pypy' python compiler, follow the link https://doc.pypy.org/en/latest/install.html
	2. Install all required packages mentioned in requirements.txt file
	3. Activate conda environment if conda is used for installing pypy
	4. Execute the command : pypy main.py VNTS <instance> <timelimit>
		replace <instance> with instance name, all instance names can be found in files/instances.txt
		replace <timelimit> with timelimit in seconds. 1 hour = 3600
